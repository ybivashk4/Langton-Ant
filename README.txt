Для запуска меню нужно зайти в папку menu и запустить menu.c в IDE, поддерживающей компиляцию c. Далее скомпилировать и открыть файл menu.exe 

После ввода нужной информации необходимо вернуться в папку "kursach" и запустить файл Langton_ant.dev 

Если в IDE не появился файл main.c - запустить этот файл в проекте. Далее скомпилировать и запустить

Чтобы начать взаимодействовать с программой нужно открыть файл menu.exe находящийся по пути menu/menu.exe и начать взаимодействие с консолью.

Далее после сохранения в главный файл (цифры 7 и 8) нужно вернутся в корневую папку и запустить Langton_ant.exe чтобы увидеть симуляцию.
